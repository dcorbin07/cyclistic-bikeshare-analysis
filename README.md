# Cyclistic Bike-Share Analysis

**Google Data Analytics Professional Certificate Capstone | March 2025**

## Overview
Analyzed 788,189 ride records from Divvy's Q1 2019 and Q1 2020 datasets to identify behavioral differences between casual riders and annual members, with the goal of informing a membership conversion strategy.

## Tools Used
- R (tidyverse) — data cleaning, wrangling, and analysis
- SQL — exploratory queries and aggregation
- Google Sheets — summary statistics and pivot analysis
- Tableau — data visualization

## Key Finding
Casual riders averaged 5,478 seconds (~91 min) per ride vs. 822 seconds (~14 min) for members — a 6.7x difference consistent across all seven days of the week, with a peak of 8,452 seconds (~141 min) on Thursday.
